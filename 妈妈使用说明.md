# 妈妈的模特图 Agent

双击 `启动模特图工具.command`，页面打开后放入衣服图片，选择画面风格，点“生成模特图”。

第一次启动会自动安装图片处理组件，等待终端窗口显示启动完成即可。之后启动不需要重复安装。

这个工具会先自动分析上传图片中的服装要素，再生成模特图。它会把颜色、材质、衣长、领子、袖口、口袋、门襟和下摆写进提示词，避免把款式改错。

手机拍摄的动态照片、景深照片或 MPO 图片会自动提取主图、校正方向并转换成标准图片，不需要手动改格式。

备注里可以再写更具体的要求，比如“口袋是斜插”或“袖口不能变直”。

如果页面显示“缺少密钥”，展开“配置 API 密钥”，分别填入硅基流动 Key 和火山引擎 Key，再点保存。不要把 key 发到聊天里。

生成结果会自动保存到 `agent_outputs` 文件夹。
